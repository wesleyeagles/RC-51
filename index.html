RC-51

Data Logger Temperatura Multiuso RC-51 (-30+70°C) 32.000 leituras. Estilo caneta, possui alarmes configuráveis, conexão USB, geração automática de relatórios em PDF. Grau de proteção IP-67

O data logger RC-51 é ideal para registrar temperaturas de produtos farmacêuticos, médicos, alimentícios e aplicações em laboratórios. Possui design estilo caneta, display LCD multifuncional, com capacidade de armazenamento de dados de 32000 leituras (podendo ser reconfigurado mediante software Elitechlog), possui alarmes configuráveis, duração da bateria de aproximadamente 2 anos, conexão USB, geração automática de relatórios em formato PDF, com intervalos de registro ajustáveis de 10s – 24hs. Possui unidade de temperatura selecionável: °C e °F. O software Elitechlog é gratuito e compatível com MAC / Win. Cada peça é submetida a rigorosa inspeção de qualidade de acordo com as normas da FDA e CE.

GARANTIA: 12 meses para defeito de fabricação
Software compatível com: Windows 7/8/10 / MAC OS
OBS: no caso de aplicações que demandem calibração por órgãos certificadores, fica a cargo do cliente a calibração.
